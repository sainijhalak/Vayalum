#  Vayalum – AI Farm Assistant

Vayalum is a Streamlit-based web application that helps users with basic farming-related information like crop guidance, weather updates, and disease detection.

## Features

- AI chatbot for farming questions  
- Weather information  
- Crop disease detection (image upload)  
- Government schemes info  
- User login and data storage  
- Voice calling support (Twilio)  

## Tech Stack

- Frontend: Streamlit  
- Backend: Python  
- Database: SQLite  
- APIs: Gemini API, OpenWeather API, Twilio  

## How to Run

```bash
git clone https://github.com/your-username/vayalum.git
cd vayalum
pip install -r requirements.txt
